# 数据源说明

## 股票列表

优先读取新浪财经沪深 A 股列表。

如果新浪列表读取失败，会尝试使用腾讯报价接口生成备用股票列表。

列表阶段会过滤：

- ST 股票
- 退市或异常名称
- 无成交股票
- 成交额低于最低门槛的股票
- 无法识别板块的股票

## 日 K 数据

优先读取新浪日 K。

如果新浪日 K 失败或返回空数据，会尝试腾讯日 K 备用源。

缓存更新时：

- 默认保留约 320 个交易日。
- 已有缓存时增量更新最近若干天。
- 更新完成后自动清理过早 K 线。

## 最新收盘日判断

根据中国时间判断目标收盘交易日：

- 交易日 15:30 前，目标为上一交易日。
- 交易日 15:30 后，目标为当日。
- 周末会回退到最近交易日。

当前判断只处理普通周末，不内置完整节假日表。

## 数据风险

公开数据源可能存在延迟、缺失、限流或接口变更。筛选结果应作为研究参考，不应作为买卖依据。
