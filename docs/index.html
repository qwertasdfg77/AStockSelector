开源 Android · A 股日线策略筛选 · 0.3.2

AStockSelector

面向日线复盘场景的手机端 A 股策略筛选工具，提供本地 SQLite 缓存、智能数据更新、预设战法筛选、今日信号留存和 GitHub 版本检测。

安装流程

下载 APK

从 GitHub Releases 下载最新版安装包。

允许安装

手机提示未知来源时，允许用于下载 APK 的浏览器或文件管理器安装。

更新缓存

首次打开后点击“智能更新并筛选”，等待本地缓存完成。

查看信号

在“今日信号”里查看命中股票、策略和规则核验。

核心能力

手机端缓存

在 Android 端联网读取 A 股列表和日 K 数据，合并到本地 SQLite 缓存，并自动清理过早数据。

策略筛选

支持年线首板、九阳蓄势、博弈K、低位启动等预设战法。

结果保留

今日信号会持久保留；新入选股票置顶并显示“新”标识，方便复盘变化。

适合谁

适合

  • 个人收盘后复盘。
  • 学习 Android、Kotlin、Compose 和 SQLite 缓存。
  • 研究日 K 技术筛选条件。

不适合

  • 盘中实时交易。
  • 自动下单或券商接口。
  • 把筛选结果当成投资建议。
