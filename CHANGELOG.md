# 更新日志

## 0.2.3

- 优化本地策略筛选的均线计算，减少重复计算 MA5/10/20/60/250。
- 缓存联网更新改为固定 worker 池，降低大量股票更新时的协程调度开销。
- HTTP 重试等待改为协程 delay，避免阻塞后台线程。
- 缓存保留窗口从约 320 个交易日回退为约 270 个交易日，减少缓存体积和读取量。
- 新增策略单元测试，覆盖年线首板、九阳蓄势、K 线不足和 ST 股票过滤。

## 0.2.2

- 智能更新改为按股票缺口增量更新，只补缺目标交易日、失败或新增股票。
- 新增缓存失败队列，失败股票会持久保存并在后续优先补读。
- 限制同一股票当天重复失败重试次数，避免异常数据拖慢整体更新。
- 设置页新增“重建缓存”按钮，可只重建 K 线缓存而不清除 App 设置。
- 缓存状态新增最新日覆盖率、失败队列数量和可重试失败数量。
- 首次和重建缓存按约 270 个交易日保留 K 线，筛选仍只读取必要窗口。

## 0.2.1

- 新增低位启动策略。
- 今日信号支持四个预设战法。
- 多战法选择改为“且”筛选。
- 同一股票多策略命中时合并显示。
- 新入选股票置顶并显示“新”标识。
- 优化本地缓存筛选窗口和候选股票并行计算。
- 增加 GitHub 程序更新检测。

## 0.1.x

- 初始 Android App。
- 支持年线首板、九阳蓄势、博弈K。
- 支持手机端联网读取和本地缓存。
- 支持智能更新并筛选。
