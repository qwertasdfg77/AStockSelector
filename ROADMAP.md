# 开发路线图

## 已完成

- Android Kotlin + Jetpack Compose 主体界面。
- 今日信号、设置两个主要页面。
- 四个预设战法：年线首板、九阳蓄势、博弈K、低位启动。
- 手机端联网读取 A 股列表和日 K 数据。
- 新浪主数据源、腾讯备用数据源。
- 本地 SQLite 缓存和智能更新。
- 缓存最新判断、增量更新、过期 K 线清理。
- 筛选结果持久化、新入选股票置顶。
- 程序更新检测。
- GitHub Actions 自动构建校验。

## 优先计划

- 增加更多中文说明和截图。
- 增加可配置的策略参数导入/导出。
- 增加本地筛选结果导出 CSV。
- 增加 K 线详情页。
- 增加更多数据质量检查。

## 暂不计划

- 自动交易。
- 券商接口。
- 实时分时盯盘。
- 收费服务端。
- 收集用户数据。
