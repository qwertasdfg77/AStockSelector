# 隐私说明

AStockSelector 当前版本为本地优先的 Android 应用。

## 本地数据

App 会在设备本地保存：

- 股票列表和日 K 缓存数据库 `market_cache.db`
- 最近一次筛选结果
- 已选择的战法
- 更新状态和缓存状态

这些数据默认保存在 App 私有目录中。

## 网络访问

App 会访问公开行情和更新地址：

- 新浪财经接口，用于读取 A 股列表和日 K 数据。
- 腾讯行情接口，作为备用数据源。
- GitHub raw 地址，用于检测 App 更新。

## 不收集的信息

当前项目没有服务端，也不会主动上传：

- 用户身份信息
- 手机通讯录、位置、相册等个人数据
- 用户本地筛选结果
- 本地缓存数据库

## Token

工程中保留了 Tushare 客户端骨架，但当前主要流程不需要用户上传 Token。请不要把自己的 Token 写进源码或公开仓库。
